Для восстановления данных из файла

Сохраните файл в формате .txt в директории с:/demo/vkt

В форме ниже введите название клиента. Его можно скопировать из сохраненного файла после двоеточия. Пример: "Абонент:Восточный луч"

    Пример: из строки "Абонент:Восточный луч" выбираем "Восточный луч" и копируем в форму ниже
